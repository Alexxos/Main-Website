Css Units

Relatives Units

  1. Percentages set at 50% of width

  2. EM's are relative units. 1em equals the font-size of the parent 2 em is twice the font size

  3. REM. Relative to the root html element's font size. if the root font-size is 2-px, 2rem is 40px, etc. Here is 2 REM

  4. View Height & View Width. Usefull with backgrounds.

    • 100vw is 100% of the width of the viewport (screen)
    • 100vh is 100% of height of the viewport

Good pratices

  • px - avoid for font-sizes. Use px for small details like borders and shadows
  • rem/em : go to choice for font-size and padding/margin
  • % : useful for defining layouts and widths
  • vh/vm : used for larger layout concerns
