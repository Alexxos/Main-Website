Css Selectors

Css selectors link

  • * The star symbol will target every single element on the page.
  • # ID selectors are rigid and don't allow for reuse
  • . This is a class selector,you can target multiple elements
  • X Y the descendant selector. When you need to be more specific with your selectors, you use these.
  • X target all elements on a page, according to their type
  • X:visited and X:link :link pseudo-class to target all anchors tags which have /yet to be clicked on.
  • class="sign"X + Y It will select only the element that is immediately preceded by the former element.
  • X > Y will only select direct children
  • X ~ Yany Y elements, as long as they follow a X.
  • X:hover
  • X:checked
    • This pseudo class will only target a user interface element that has been checked - like a radio button, or checkbox.
  • input[type="text"]selection by attributes
